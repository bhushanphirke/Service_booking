## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- PHPMailer

## Features
- User Registration & Login
- Service Booking
- Email Notification
- Admin Panel

## Setup Instructions
1. Import database (.sql file)
2. Rename config.example.php to config.php
3. Add database credentials
4. Run on localhost or live server
